# Project Brief

Penn Builder Compact is a self-contained website builder for creating a Penn-styled landing page or microsite preview. The goal is to let a non-technical editor assemble sections, choose visual styling, set navigation groups, preview the result, and download a portable ZIP export.

## Product Goals

- Make page composition fast and visual.
- Keep the implementation portable as plain HTML, CSS, and JavaScript.
- Support a Penn-inspired visual system without hard-coding every decision.
- Let editors control content, images, links, section order, and grouped navigation.
- Export a usable static site bundle with images included.

## Primary User

An editor or designer building a structured Penn-themed page who needs to preview, tweak, export, and hand off static HTML.

## Design Direction

The project currently uses:

- Penn Classic theme colors by default
- Roboto as the primary typeface
- EB Garamond as the secondary typeface
- University of Pennsylvania logo assets
- Editorial layouts for events, quotes, cards, galleries, and hero content

## Non-Goals

- No CMS backend
- No authentication
- No database
- No dependency install
- No production build process
